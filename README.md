# Employee Data Analysis & Interactive Dashboard

## Project Overview
This project involves a comprehensive analysis of employee data using Excel. The goal was to transform raw data into a functional, interactive dashboard that provides insights into workforce distribution, salary trends, and departmental demographics.

## Tools Used
* **Microsoft Excel:** Core analysis and visualization.
* **Power Query (ETL):** Used for cleaning, extracting, and transforming the raw employee dataset (handling nulls, formatting dates, and organizing locations).
* **Pivot Tables:** For data aggregation.
* **Excel Dashboards:** Created an interactive interface using Slicers and Timelines.

## Key Insights & Features
* **ETL Process:** Cleaned a dataset of 450+ employees, ensuring consistency across departments like Accounting, Engineering, and Sales.
* **Salary Analysis:** Visualized salary distributions across different years and departments.
* **Workforce Demographics:** Analyzed gender distribution and headcount across various global locations (USA, NZ).
* **Interactivity:** Users can filter the entire dashboard by Department or Gender to see real-time updates.

## How to View the Project
1. Download the `Employee_Data_Analysis.xlsx` file from this repository.
2. Open the file in Microsoft Excel.
3. Use the Slicers on the 'Dashboard' sheet to interact with the data.
